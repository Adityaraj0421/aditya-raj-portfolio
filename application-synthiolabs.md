# SynthioLabs — Design Engineer application note

I designed the human-control layer for an AI concierge before I designed its polish.

At Find My Genie, the agent coordinated outreach across more than 20 vendors at once. Because availability changed asynchronously and the agent could misunderstand context, I designed and implemented traceable workflow states, intervention controls, verification steps, and operator recovery paths across four production React/Next.js portals. The domain was hospitality rather than healthcare, but the safety principle maps directly to regulated AI: consequential automation should remain inspectable, uncertainty should be visible, and a human should have a clear path to intervene.

I now own ContentOS at Fobet Media end to end. I architected its multi-stage LLM pipeline—intake, scraping, ranking, and script generation—and built the production interface on Next.js, Supabase, and Gemini. Working across product and implementation lets me respond to real system behavior rather than treating design as a handoff.

I also created Naksha Studio, an open-source design system for AI coding tools with 26 specialist roles, 62+ commands, 15,000+ lines of design knowledge, and 300+ GitHub stars. It reflects how I work: investigate the system, encode repeatable judgment, and stay close to the code that ships.

SynthioLabs is compelling to me because the interface is part of the safety model. Voice and video interactions for patients and healthcare professionals need clarity, traceability, empathy, accessible recovery, and strong human escalation—not just a polished conversational surface. I would bring end-to-end product ownership, visual craft, React/TypeScript implementation, and hands-on experience designing controls around non-deterministic AI behavior.
